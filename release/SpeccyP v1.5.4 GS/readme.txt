GSP v1.5.4 [24.02.2026]
Эмулятор General Sound Pico (GSP).
+ SpeccyP v1.5.4 GS для него.

Эмулируются:
# GeneralSound;
# TurboSound (вместо TS на основной плате);
# Z-controler Контроллер SD карты 
    (используется разъём SD на плате дополнительного мурмулятора);
# MIDI упрощенная тестовая версия 
  подключение MiDI по версии ShamaZX (порты 0xA1CF, 0xA0CF)
  настроенный плейер и TRD с midi файлами прилагается 
  плейер поддерживает Z-Controller)

Работает на двух платах Мурмулятора ПЕРВОЙ ревизии (MURM1)

1- Основная плата с прошивкой  SpeccyP_154_GS_m1p1.uf2 (RP2040) 
                           или SpeccyP_154_GS_m1p2.uf2 (RP2350)

2- Дополнительная (MURM1 с I2S звуком!) ТОЛЬКО для RP2350 с прошивками GSP:
  * GSP-PSRAM_154_m1p2.uf2 RP2350 с "бутербродной" PSRAM  CS=19 (GS доступно 2048 Кб) 
  * GSP-NO_PSRAM_154_m1p2.uf2 RP2350 БЕЗ PSRAM (GS доступно 160 Кб)
 
Соединены проводами МИНИМАЛЬНОЙ длины 

1.ПЛАТА MAIN    2.ПЛАТА GS
GPIO 26         GPIO 15
GPIO 27         GPIO 14
GPIO 28         GPIO 16
RUN             RUN
GND             GND

Соответственно на платы подать питание +5V
желательно от одного источника 
я от USB компа всё пока включаю ;)

1.ПЛАТА MAIN  перемычки на звук с платы 26,27,28,29 должны быть убраны!
2.ПЛАТА GS перемычки с платы 14,15,16,17 на джойстик должны быть убрать!


При включении мурмуляторы должны синхронизироваться с друг другом,
если при старте на экране надпись синим цветом «PicoBus connected...» 
то мурмуляторы не могут синхронизироваться.
 Помогает нажатие кнопки RUN любой из плат,
 Если пины RUN двух pico соединенны 
 то как правило синхронизация происходит автоматически


Работает регулировка громкости [F7] и [F8] а также buster i2s  в разумных пределах

NEW Повышенно питание ядра pico до 1.4В из-за не работы некоторых модулей на 378МГц 1.3В
